## Local Setup



```bash 
git https://github.com/xenon-project/xenon-project.github.io.git
```

```bash 
cd xenon-project.github.io
```

```bash 
pnpm i
```

```bash 
pnpm run docs:dev
```

```bash 
pnpm run docs:build 
```
